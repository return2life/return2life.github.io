```code
# 1. 아치 리눅스 설치
## archlinux.org  /  aur.archlinux.org
1. 윈도우 우선설치..
2. 윈도우 기본세팅 완료
3. 디스크메니저 > 리눅스 공간 할당
4. 윈도우 복구시점 만들기(선택)
5. 리눅스 설치
5-1. iwctl , device list(show),
    station "wlan0" get-networks
    station "wlan0" connet "SSID"
5-2. pacman -Sy
5-3. pacman -S archlinux-keyring
5-4. pacman -S archinstall
6. lsblk > 리눅스 설치 파티션 확인
7. cfdisk /dev/"리눅스설치할 물리!! 파티션"
8. 파티션 할당하기
8-1. 1gb (EFI system,부트로더용) , 나머지(linux file sysstem), 램사이즈(swap)
9. file system format
    mkfs.fat -FAT32 /dev/"nvme0n1p??"(부트 1g)
    mkfs.ext4 /dev/"nvme0n1p??"(최상위 나머지g)
10. mount
    mount /dev/"nvme0n1p??" /mnt (최상위 나머지g)
    mkdir /mnt/boot
    mount /dev/"nvme0n1p??" /mnt/boot(부트로더용 1g)
11. archinstall
    메뉴보고 알아서 설정
    디스크설정>파티션>pre-mount ~~~~
    : /mnt
    ; package- nano git flatpak fastfetch base base-devel zsh
12. after install(post-install configuration)
    >pacman -S grub efibootmgr dosfstools mtools
    >grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB (부팅메뉴명)
    >grub-mkconfig -o /boot/grub/grub.cfg
    >umount -lR /mnt
    >shutdown now   or reboot now
    =======================   option ===============
    linux boot> terminal
    >sudo nano /etc/default/grub
    : #GRUB_DISABLE_OS_PROBRT=fasle 주석해제 다른 OS 감지
    저장하고 나가기
    >sudo pacman -S os-prober
    >sudo grub-mkconfig -o /boot/grub/grub.cfg   ; grub설정파일재생성하면 프로브가 알아서 메뉴추가함
    안보이게 하려고 프로브 설치및 설정 안햇음.
```
