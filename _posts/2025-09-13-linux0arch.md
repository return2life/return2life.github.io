# 1. 아치 리눅스 설치
## archlinux.org  /  aur.archlinux.org
1. 윈도우 우선설치..
2. 윈도우 기본세팅 완료
3. 디스크메니저 > 리눅스 공간 할당
4. (선택) 윈도우 복구시점 만들기 또는 백업프로그램사용..
5. 리눅스 설치
5-1. iwctl , device list(show),
    station "wlan0" get-networks
    station "wlan0" connect "SSID"
    연결되면 exit
5-2. pacman -Sy
5-3. pacman -S archlinux-keyring
5-4. pacman -S archinstall
6. lsblk > 리눅스 설치 파티션 확인
7. cfdisk /dev/"리눅스설치할 물리!! 파티션"
8. 파티션 할당하기
8-1. 1gb (EFI system,부트로더용) , 나머지(linux file sysstem), //램사이즈(swap)
9. file system format
    mkfs.fat -FAT32 /dev/"nvme0n1p??"(부트 1gb)
    mkfs.ext4 /dev/"nvme0n1p??"(최상위 나머지gb)
10. mount
    mount /dev/"nvme0n1p??" /mnt (최상위 나머지gb)
    mkdir /mnt/boot
    mount /dev/"nvme0n1p??" /mnt/boot(부트로더용 1gb)
11. archinstall
    메뉴보고 알아서 설정
    디스크설정>파티션>pre-mount ~~~~
    : /mnt
    ; package- nano git flatpak fastfetch base base-devel zsh
12. after install(post-install configuration)
    >pacman -S grub efibootmgr dosfstools mtools
    >grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB (부팅메뉴명)
    >grub-mkconfig -o /boot/grub/grub.cfg
    >umount -lR /mnt
    >shutdown now   or reboot now
13. hangul
    >fcitx5-im, fcitx5-hangul 설치 //fcitx5-configtools
    >fcitx5-im 패키기 안에 fcitx5 와 fcitx5-configtools 포함되 있을거야?;;    
    > 입력기에서 "한글"선택되잇어야 한글사용됨
    > 한글은 fcitx 와 nimf 두개를 병행해서 쓰기...
    ; fcitx5: 메인 입력기 프레임워크
    ; fcitx5-hangul: 한글 입력 지원
    ; fcitx5-configtool: 설정 GUI
    !! ibus는 한글입력 상태에서 띄어쓰기가 안된다..
    !! 한글편집기(?)에선 nimf 입력기를 사용한다.
    !! 한글입력이 안될시 " /opt/hnc/hoffice11/Bin/ "(설치경로)내에 qt 폴더를 qt.bak 등으로 이름바꾸기 한다.
14. 프롬프트 꾸미기
    > oh-my-zsh
    : sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
    : .zshrc에서 플러그인을 활성화합니다.
      ZSH_THEME="agnoster"
      plugins=( git zsh-autosuggestions zsh-syntax-hightlightin )
    > plugin : autosuggest, syntax-highlight 두개만 설치 딴건 나랑 안맞다
    : git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
    : git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
15. timeshift
    > backup&restore
    : 시스템만 백업복구된다.. 나머지 수동으로 하던지 rsync등을 써야하는데 할 줄 모른다..
